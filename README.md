# Data Classification Using AI

## Description
This project is a basic machine learning classification model developed in Python using the K-Nearest Neighbors (KNN) algorithm. It demonstrates the fundamentals of supervised learning by loading a dataset, splitting it into training and testing sets, training a classification model, and evaluating its performance.

## Features
- Loads the Iris dataset.
- Displays the dataset.
- Splits the dataset into training and testing sets.
- Trains a K-Nearest Neighbors (KNN) classifier.
- Predicts the class of test data.
- Calculates and displays the model accuracy.
- Generates a classification report.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Dataset
This project uses the built-in Iris dataset provided by the Scikit-learn library. The dataset contains 150 flower samples from three different Iris species and is commonly used for machine learning classification.

## How It Works
1. Load the Iris dataset.
2. Prepare the feature and target data.
3. Split the data into training and testing sets.
4. Train the KNN classification model.
5. Predict the test data.
6. Evaluate the model using accuracy and a classification report.

- Model accuracy.
